    01. Desafios Básicos:
1) Crie um programa que receba o nome e a idade do usuário e imprima uma mensagem com essas informações.
2) Faça um programa que peça ao usuário para digitar dois números e imprima a soma deles.
3) Crie um programa que converta uma temperatura de Celsius para Fahrenheit.
4) Faça um programa que solicite o raio de um círculo e calcule sua área.
5) Crie um programa que receba um número inteiro e verifique se é par ou ímpar.

    02. Desafios de Strings:
1) Faça um programa que peça ao usuário seu nome e sobrenome separadamente, e depois imprima-os juntos.
2) Crie um programa que substitua todas as ocorrências de uma letra em uma frase por outra letra especificada pelo usuário.
3) Faça um programa que verifique se uma palavra digitada pelo usuário é um palíndromo.
4) Crie um programa que conte quantas vezes uma determinada palavra aparece em uma frase.
5) Faça um programa que receba uma frase e a imprima em caixa alta e caixa baixa.
